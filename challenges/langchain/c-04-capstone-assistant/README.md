# c-04-capstone-assistant

Production-shaped assistant: route to RAG vs direct, gate sensitive verbs behind HITL, keep a trace, enforce a budget.

## Validator

From repo root:

```bash
npm run challenge -- capstone-assistant --track langchain
```

Or from this package directory:

```bash
cd challenges/langchain/c-04-capstone-assistant
uv run python tests/validate.py
```
