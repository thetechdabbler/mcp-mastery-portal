# Course design

## Goals

- Teach MCP from first principles to production governance.
- Separate **content** (MDX under `content/`) from **application** (`app/`, `components/`).
- Keep v1 local-first: `localStorage` progress, no database.

## Pedagogy

- Each chapter: objectives, diagram-first mental model, walkthrough, Hall of Shame, production framing, quiz, references.
- Labs are runnable Node workspaces mirroring real SDK usage.
- Challenges automate validation with the same MCP client primitives learners use.
